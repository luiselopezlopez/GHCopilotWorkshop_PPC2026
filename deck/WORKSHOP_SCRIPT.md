# Workshop Script

## Slide 1

Titulo: GitHub Copilot Hands-on Lab

Mensaje: hoy vamos a usar Copilot para entender, planificar, implementar y customizar.

## Slide 2

Mapa de la sesion:

1. Ask, Plan y Agent sobre una app rota.
2. Prompts, Instructions, Skills y Agents en el repo.

## Slide 3

Caso del dia:

Una aplicacion web de notas, local y monolitica, con un bug que impide el arranque.

## Slide 4

Ask, Plan y Agent:

- Ask para entender.
- Plan para acotar.
- Agent para ejecutar.

## Slide 5

Bug del laboratorio 1:

- la app no arranca,
- el objetivo es entender la causa,
- el arreglo debe ser pequeño y verificable.

## Slide 6

Customizaciones:

- prompt: una tarea,
- instructions: reglas persistentes,
- skill: procedimiento,
- agent: rol especializado.

## Slide 7

Cierre:

- usa el modo Ask para entender antes de tocar codigo,
- obliga a Copilot a proponer planes pequenos,
- encapsula patrones recurrentes en customizaciones.
